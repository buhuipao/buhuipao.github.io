Linux 下特殊变量：（$N $* $@ $! $? $0 $$ !$ ）

$N:表示第N个参数，例如$1,$2,$3

$*：表示把所有参数作为一个字符串

$@:表示把每个参数单独作为一个字符串；

$!:表示刚执行的命令的pid；

$?:表示退出数，0～254，0表示命令成功执行；

$0:表示表示命令本身；

$$:表示此进程的pid，记得与$!区别；

!$:表示上一次命令的参数；

${#var}:表示$var的字符数；
